# santa-clothes-backend
> ### *About*
캡스톤 디자인(졸업 작품)으로 제작하고 있는 프로젝트로, 현재 Back-End 웹 서버 개발 및 리팩토링을 맡고 있습니다.

</br>

</br>

> ### *Santa Clothes?*
<img width="700" alt="?????????? 2023-04-25 ???? 11 59 31" src="https://user-images.githubusercontent.com/106216912/234495393-1ec7ebaf-458f-493d-ae00-b36d9aada11f.png">

*Santa Clothes는 어려운 이웃들에게 도움이 되자는 취지로 만들어진 프로젝트로, 중고 의류  거래 중계 플랫폼 입니다.*

*사용자는 게시글과 채팅을 통해 의류를 직접 주고 받을 수 있고, 관리자에게 서로 간의 중계를 요청할 수도 있습니다.*

*또한, 기부를 통해 쌓인 포인트로 실생활에 필요한 물품을 구매할 수 있습니다.*

*나눔의 생각은 있지만, 직접 실천하지 못하고 생각만 하는 사람들을 위해 사용하기 쉬운 웹 서비스를 제공합니다.*
